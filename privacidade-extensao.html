Política de Privacidade — Extensão “Captação · Garimpo LinkedIn”

Última atualização: 01/07/2026

Esta extensão é uma ferramenta interna de produtividade usada por uma equipe de captação para organizar seus próprios contatos e conversas do LinkedIn dentro de um CRM privado. Ela é de uso restrito e disparo manual.

Que dados a extensão acessa

  • Somente dados que o próprio usuário já enxerga logado na sua conta do LinkedIn: nome, cargo/empresa, foto de perfil, e o conteúdo das conversas na sua Caixa de entrada (prévia, horário, últimas mensagens).
  • A leitura acontece apenas quando o usuário clica em “Sincronizar” / “Captar” / “Atualizar este lead”. Nada é lido em segundo plano.

Como os dados são usados

  • Os dados capturados são enviados exclusivamente para o CRM privado do próprio usuário (banco de dados Supabase da organização), para popular a ficha do lead e o histórico de contato.
  • Alternativamente, os dados são copiados para a área de transferência para o usuário colar no seu próprio CRM.

Compartilhamento

Nenhum dado é compartilhado, vendido ou enviado a terceiros. Não há rastreamento, anúncios ou analytics. O único destino dos dados é o banco de dados privado da própria organização que usa a ferramenta.

Armazenamento

A extensão guarda localmente (no chrome.storage do navegador) apenas o token de sessão do CRM e os modelos de mensagem do usuário, para não pedir login a cada uso. Os dados dos leads ficam no banco de dados privado da organização.

Permissões

  • clipboardWrite: copiar os dados capturados para você colar no CRM.
  • storage: guardar o login do CRM e seus modelos de mensagem.
  • Acesso a linkedin.com: ler a página que você está vendo para capturar o contato/conversa.
  • Acesso ao domínio do Supabase: gravar o lead no seu CRM privado.
