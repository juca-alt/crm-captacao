Política de Privacidade — Extensão “Captação · WhatsApp → CRM”

Última atualização: 21/07/2026

Esta extensão é uma ferramenta interna de produtividade usada por uma equipe de captação para organizar seus próprios contatos dentro de um CRM privado. Ela é de uso restrito e somente leitura no WhatsApp: não envia mensagens, não clica, não automatiza nenhuma ação no WhatsApp Web.

Que dados a extensão acessa

  • Somente o que o próprio usuário já enxerga logado no seu WhatsApp Web: o nome e o telefone do contato da conversa aberta, para localizar (ou criar) a ficha desse contato no CRM.
  • A extensão não lê o conteúdo das mensagens e não coleta nada em segundo plano — apenas detecta qual conversa está aberta para mostrar o card correspondente.

Como os dados são usados

  • Os dados digitados no card (status, observações, cargo, empresa etc.) são enviados exclusivamente para o CRM privado do próprio usuário (banco de dados Supabase da organização), para popular a ficha do lead.

Compartilhamento

Nenhum dado é compartilhado, vendido ou enviado a terceiros. Não há rastreamento, anúncios ou analytics. O único destino dos dados é o banco de dados privado da própria organização que usa a ferramenta.

Armazenamento

A extensão guarda localmente (no chrome.storage do navegador) apenas o token de sessão do CRM, para não pedir login a cada uso. Os dados dos leads ficam no banco de dados privado da organização.

Permissões

  • storage: guardar o login do CRM.
  • Acesso a web.whatsapp.com: ler o nome/telefone da conversa que você está vendo, para mostrar o card do lead.
  • Acesso ao domínio do Supabase: ler e gravar o lead no seu CRM privado.
